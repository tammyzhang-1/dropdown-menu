# Dropdown Menu

## Prerequisites
Make sure you have the following installed:
- Node.js (version 16 or higher recommended)
- npm or pnpm or Yarn

## Cloning the App
```
git clone https://github.com/tammyzhang-1/dropdown-menu.git
cd dropdown-menu
npm install
```

## Building and Running the App
```
npm run dev
npm run build
```
